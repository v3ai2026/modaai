# React Demo

This is a demo application showcasing the V3 AI UI React components.

## Run the demo

```bash
# From the root directory
pnpm install
pnpm lib:build
cd examples/react-demo
pnpm dev
```

Then open http://localhost:3001 in your browser.

## Features Demonstrated

- **Icons**: 40+ SVG icons with customizable size and color
- **Animations**: Motion components with smooth transitions
- **Buttons**: Multiple variants and sizes
- **Dialog**: Modal dialog with customizable content
- **Hooks**: useLocalStorage, useToggle
